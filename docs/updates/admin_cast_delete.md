# /admin/casts/[castId] でキャスト削除を可能に

- キャスト詳細ページの最下部に削除セクションを追加し、ダイアログ確認後に登録済みキャストを削除できるようにしました。削除完了後はキャスト一覧へリダイレクトされます。
- 削除実行時は認証状態を再確認し、エラー内容をページ内で利用者に通知します。
- API に `DELETE /api/admin/casts/[id]` を実装し、削除時に `cast_social_links` と `cast_follower_snapshots` を先にクリーンアップしてから `casts` テーブル本体を削除するため、関連データが残りません。
