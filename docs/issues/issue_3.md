# エリア選択フローとキャスト一覧ページの開発

# 目的
エリアから探すボタンを押下した後のページ遷移を開発する。

# 要件
1. トップページに表示されているエリアを探すコンポーネント内から都道府県、繁華街情報を削除する（feature_issue_1の状態に戻す）
2. エリアから探すボタンを押下すると、/todofuken-choice画面に遷移する
3. /todofuken-choice画面でareaテーブルに存在する都道府県を選択することができる
4. 都道府県を選択すると、/downtown-choice画面に遷移する
5. 繁華街名を選択すると、当該繁華街の店舗に勤務するキャストのキャストカードが10表示される。
6. 11件以上キャストが存在する繁華街のキャスト一覧ページはページネーションで11件~20件目を表示することができる。
7. 21件以上存在する場合は、6に同様でページネーションを使って閲覧できるようにする。
8. キャストカードはトップページのキャストカードと同様のカードを表示する。

# スコープ外
- DBからのキャスト情報の取得は不要。各繁華街に静的なモックデータを31件表示すること。
