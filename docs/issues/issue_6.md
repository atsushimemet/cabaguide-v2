# 管理者ページとキャスト登録画面を作成する

# 目的
- スマホでキャスト登録できるようにする

# 要件
- /admin/loginにアクセスできる
- 当該画面においてパスワードを入力できる
- .envファイルに記載されているADMIN_PASSWORDを入力すると/adminに遷移する
- /adminページには店舗登録導線、キャスト登録導線（/admin/casts）がある。
- 店舗登録導線では以下の値を登録できる。
  - 繁華街名（必須：area_idと繁華街名のセットのプルダウンから選択する）
  - 店舗名（必須：テキスト入力）
  - Google マップの URL（必須：テキスト入力）
  - 電話番号（必須：テキスト入力。ハイフン含む）
  - 指名料金（任意：数値入力: スマホで入力できるように1000~10000をピッカーで指定できるようにする。10000刻み。）
  - サービス料パーセンテージ（任意：小数入力：スマホで入力できるように0.10~から~0.50をピッカーで指定できるようにする。0.05刻み。）
  - 消費税は10%固定のため入力項目は不要（自動計算）
  - 軽ドリンク/シャンパンの価格は固定想定（1杯2,000円・1本25,000円）のため入力項目は不要。
  - タイムスロット（必須：数値入力: スマホで入力できるように20~24をピッカーで指定できるようにする。1刻み。）
  - メイン料金（必須：数値入力: スマホで入力できるように1000~20000をピッカーで指定できるようにする。1000刻み。）
  - VIP料金（必須：数値入力: スマホで入力できるように5000~50000をピッカーで指定できるようにする。1000刻み。）
- キャスト登録導線では以下の値を登録できる。
  - 店舗名（必須：店舗idと店舗名のセットのプルダウンから選択する）
  - キャスト名（必須：テキスト入力）
  - 年齢（任意：数値入力: スマホで入力できるように18~40をピッカーで指定できるようにする。）
  - image_url メイン画像 URL（任意：当初は入力しない。MAUが増加してからキャストに許諾を得て指定する。）
- 登録されたキャストは/admin/castsにキャストカードで表示される。
- 当該動線でキャストカードを押下すると、/admin/casts/[:id]に遷移する
- 当該画面では以下の値を入力・反映できる
  - instagramフォロワー数
  - tiktokフォロワー数
- 当該画面では登録時の時間でcaptured_atが登録される
