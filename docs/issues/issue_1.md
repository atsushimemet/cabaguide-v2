# トップページの開発

# 目的
cabaguideのトップページを開発する

# 要件
## 機能要件
1. ヘッダーの1行目にcabaguideと記載されている
2. cabaguideを押下するとホーム画面に遷移する
3. ヘッダーの2行目に1行目より小さい文字で「この子でよかった」を選べるネット案内所と記載されている
4. bodyの最上部にエリアから探すボタン（TOP）が設置されている
5. エリアから探すボタン（TOP）の下に広告エリア（TOP）が設置されている
6. 広告エリア（TOP）の下に以下のトップキャストカードが並んでいる
   1. 北海道の1位のキャスト
   2. 宮城県の1位のキャスト
   3. 東京都の1位のキャスト
   4. 埼玉県の1位のキャスト
   5. 神奈川県の1位のキャスト
   6. 千葉県の1位のキャスト
   7. 愛知県の1位のキャスト
   8. 大阪府の1位のキャスト
   9. 京都府の1位のキャスト
   10. 福岡県の1位のキャスト
7.  各キャストカードにはキャスト画像、キャスト名、総フォロワー数、店舗名が表示されている。
8.  キャスト画像、キャスト名をクリックするとキャスト詳細ページに遷移する。
9.  店舗名をクリックすると店舗詳細ページに遷移する。
10. トップキャストカードの下に広告エリア（BOTTOM）が表示されている。
11. 広告エリア（BOTTOM）の下にフッターが設置されている。
12. フッターには以下導線が設置されている。
    1.  ホーム
    2.  これはなに？
    3.  広告プラン
    4.  更新情報
    5.  FAQ
    6.  お問い合わせ
    7.  利用規約
    8.  障害情報

## UI要件
1. フォント
   1. 夜の街の雰囲気を想起させる装飾的なサンセリフ系を使用する
   2. 見出しは太め、本文は可読性を優先したシンプルな書体にする
2. 色
   1. ベースカラーは濃い紺や黒
   2. アクセントカラーにピンクや紫の鮮やかな光沢色を使用する
   3. 発光しているように見えるグラデーションを使う
3. 効果
   1. 文字やボタンに外側の光彩を薄く付ける
   2. ホバー時に色の輝きが強くなるアニメーションを付ける
4. 画像や装飾
   1. 背景にぼんやりした光の粒やネオンの光跡を入れる
   2. 直線ではなく柔らかい曲線を多めに使う

# スコープ外
- 要件に記載の、キャスト詳細ページへの遷移、店舗詳細ページへの遷移、フッター要素押下時の遷移はスコープ外。クリックできるようにだけしておく。
