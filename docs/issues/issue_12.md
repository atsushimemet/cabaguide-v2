# キャスト詳細ページの料金を概算であると明確に示し、詳細は公式HPに遷移させる

# 目的
- キャスト所属の店舗毎に料金体系が大きく異なるため、本プロダクトでは料金概算を表示することを重視する要件とする。

## 背景
- キャバクラの料金体系は店舗ごとに大きく異なる。
- 例えば桃李は分単位で細かく区切られた時間帯料金を採用し、20時台前半から後半で価格が変動する特徴がある。
- 一方、美人茶屋新宿はメイン、セミVIP、ロイヤルVIP、ロイヤルスウィートといった席種ごとに料金が分岐する点に特徴がある。
- Club UNJOURでは、一般席とVIP席に加えて、VIP個室の利用料金が人数に応じて複数設定されている点に特徴がある。
- これら3店舗の比較からも、キャバクラの料金は店舗ごとに設計思想が大きく異なり、単一のスキーマで完全に統一的に管理することが困難である。

# 要件
- キャスト詳細ページの「人数・時間を調整して予算感を掴みましょう。」を「人数・時間を調整して予算感を掴みましょう。開始時間から連続した2時間分の料金を参照します。料金概算のため詳細は店舗ページをご確認ください。」という表現に変える。
- 店舗ページには当該キャストの所属する店舗ページリンクを貼る。
- 算出されるESTIMATEを5000円単位で四捨五入し、概算値を5000円単位で表示する。
