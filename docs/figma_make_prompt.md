# figma_make_prompt.md

## 目的
Figma Make に入力し、cabaguide の既存UIを踏まえたアップデート版デザインを生成するためのプロンプト。  
*キーワード*: ネオンサイバー、夜景、キャバクラ専門メディア、フォロワーランキング、ハイコントラスト、信頼感と遊び心の両立。

## ブランド/トーンの指示
- 背景は常に #050312 ～ #03010a あたりの漆黒に、シアン/フューシャ系のソフトグローを足した「夜の繁華街」世界観。
- タイポグラフィは UI 全体で Geist Sans をベースにした幾何学サンセリフ。見出しは 32-48px、本文 14-16px。英語ラベルはトラッキング広め (0.3em)。
- ロゴ「cabaguide」は小文字・トラッキング広め・ネオンアウトライン。下に日本語タグライン「「この子でよかった」と思える夜へ」をタイプライター風に表示。
- カラーパレット  
  - 背景: #050312, #03010a  
  - プライマリグロー: #f472b6, #c084fc, #a5b4fc  
  - セカンダリアクセント: #67e8f9, #7dd3fc, #c8f0ff  
  - 警告/強調: #fcd34d (王冠), #bef264 (差し色)  
  - テキスト: #f7f5ff (メイン), #b8b4d3 (サブラベル)
- ボタン/カードはネオンフレーム、グリッチ、レーザー演出を持つ。陰影はドロップシャドウではなくグローを優先。

## 共通レイアウト
1. **PageFrame**
   - 中央 1200px コンテナ。上下余白大。
   - ヘッダーはネオンレーザーフレーム。ロゴ＋タグライン＋ページ種別のラベル (STEP1 等) を載せる。
   - フッターは 9 リンクを 4 カラムに並べる。淡い罫線と大文字ラベル「FOOTER」が先頭。
   - バックグラウンドに 2～3 個の円形グラデーション(フューシャ、シアン)をぼかし、下端は黒グラデの幕。
2. **LoadingScreen**
   - フルスクリーン固定。ロゴがRGBチャンネルでグリッチし、最後にズームアウトしながら消える。
3. **セクション共通要素**
   - 各セクションは border-y border-white/15 の罫線で区切る。内側余白は上下40px以上。
   - セクション先頭には `text-xs uppercase tracking-[0.4em]` のラベル (TOP CAST, STEP 1 など)。

## 主要コンポーネント
1. **AreaSearchCTA**
   - 左: 「エリアから探す」見出し + サブコピー。右: 円形グラデボタン。スクロールでスライドインする演出。
   - ボタンは `glitch-button`。ホバーするとRGBがズレる。
2. **AdBanner**
   - 横長。左にラベル/見出し/説明、右に丸角ボタン。「トップ」「ボトム」で左右スライドインアニメ差分あり。
3. **CastCard**
   - 3:4 の縦型イメージ。右上にアクセントバッジ (都道府県)、左上に王冠絵文字(順位1-3)。  
   - 下部にグラデオーバーレイ＋キャスト名(フォントローテーション: serif, Yuji, Dela)。表示後は1秒周期でフォント切替。  
   - 下段に総フォロワー数＋店舗名。カード全体にグリッチ枠（neon-laser-frame）。
4. **BudgetCalculator**
   - キャスト詳細右に置く。上部: BUDGET ラベル＋「2時間滞在の概算」。  
   - 展開式の注意書き、開始時間のセレクタ、概算パネル(料金内訳リスト)を持つ。
5. **Profile/SNSブロック**
   - キャスト詳細左カラム: Hero画像(640x820)、地域バッジ、名前＋年齢ラベル(数字 or 「未確認」)。  
   - SNSセクション：Instagram/TikTokアイコン、フォロワー数、人型アイコン。  
   - 店舗情報: STORE 見出し、店名リンク、Google Map、電話番号。

## ページ別の構成
### トップページ `/`
1. LoadingScreen → PageFrame。
2. AreaSearchCTA (ID=area-search)。「エリアから探す」ボタンで `/todofuken-choice` へ。
3. AdBanner(トップ枠)。
4. TOP CAST セクション: 今週のベスト10。リード文 + 更新日 + TopCastGrid (3カラム/レスポンシブ1-2-3)。
5. AdBanner(ボトム枠)。
6. AreaSearchCTA (ボーダー非表示版) を再表示。
7. 構造化データ埋め込み(内部処理だが、Figmaではラベル等で分かるように)。

### 都道府県選択 `/todofuken-choice`
- STEP1 見出し。リードコピー。
- 2カラムカード群。各カード: 都道府県名(32px) + 登録繁華街数。ホバーで白ボーダー/背景が強調。
- 下部ナビに「トップページに戻る」リンク。

### 繁華街選択 `/downtown-choice`
- STEP2 見出し (「{prefecture} の繁華街を選択」)。
- グリッドカード (2カラム) で downtown 名のみを強調。
- 下部ナビ: 都道府県一覧に戻る / トップへ戻る。

### キャスト一覧 `/casts/[downtownId]`
- ヒーロー: 「【2024年最新】{都道府県} {繁華街} キャストランキング」 + 説明 + 最終更新テキスト。
- AdBanner(繁華街プラン)を中段に挿入。
- キャストカードグリッド (10件/ページ) とランキング番号。各カードの「詳細を見る」リンクは `?from=list&page=n` を付与。
- ページネーション：現在ページ強調、ellipsis 表示。
- ページ末尾: とどふけん選択・繁華街選択・TOPリンクのCTA群。

### キャスト詳細 `/casts/[downtownId]/[castId]`
- 戻るリンク (トップ/一覧/店舗に応じて動的)。
- 2カラム: 左=キャストプロフィール、右=BudgetCalculator。
- プロフィール内訳: 画像(640x820)、居住地バッジ、名前(48px)＋年齢 (例: `25歳` / `未確認`)、SNSフォロワー（IG/TikTok/総計）3カラム、STORE情報ブロック（店名リンク, GoogleMap, 電話, store詳細へ遷移）。
- 戻り導線: デフォルト「{都道府県} {繁華街} のキャスト一覧に戻る」。

### 店舗詳細 `/stores/[storeId]`
- 戻るリンク (デフォルト: STEP選択ページ)。
- STORE セクション: 店名, 所在地ラベル, 電話, Google Map, 基本価格(指名料/サービス料/消費税), 店舗HPリンク。
- TIME SLOT セクション: 2カラム表で時間帯と料金(通常席)。
- CAST セクション: 所属キャストを CastCard で表示。0 件なら破線枠のメッセージ。

## インタラクション/アニメーション指針
- グリッチ: `glitch-button`, `cast-card-glitch` のように、表示時にRGBずれ・走査線・レーザーライン。
- スクロールトリガー: AreaSearchCTA, AdBanner(bottom)は Intersection Observer でフェードイン。
- タイプライター: ヘッダータグライン (PageFrame)。
- LoadingScreen: 3 秒。ズームして暗転。

## 表示すべきデータポイント
- キャスト: 名前, 画像URL, 年齢 (nullable), 都道府県, 繁華街名, 総フォロワー数, SNS別フォロワー, 所属店舗名/リンク。
- 店舗: 名前, 電話, Google Mapリンク, 住所ラベル, 指名料, サービス料率, タイムスロット料金, ホームページ。
- ランキング: ページネーション、順位バッジ (1〜3=👑)。
- CTA: 「エリアから探す」「広告掲載について」「トップページに戻る」などを画面下部に必ず配置。

## Figma Make への指示例
```
cabaguide というネオン系キャバクラ情報サイトの UI をリフレッシュしたい。
ダーク(#050312)背景にフューシャ/シアンのグロー、幾何学サンセリフ (Geist)、
1px の白/シアン罫線とグリッチ演出を組み合わせ、以下のページを作る:
- トップページ: AreaSearchCTA → プレミアム広告 → 今週のベスト10 → 広告 → CTA
- STEP1/STEP2 選択ページ: border-y 区切り + 2カラムのカード一覧
- キャスト一覧ページ: ランキングヒーロー + CastCardグリッド + ページネーション
- キャスト詳細ページ: 左にキャスト詳細(年齢/フォロワー/SNS/店舗)、右にBudgetCalculator
- 店舗詳細ページ: 店舗情報, タイムスロット表, 所属キャスト
各セクションは STEP ラベルや TOP CAST など大文字サブラベルを持ち、returnリンクやCTAを必ず配置する。
グリッチ/レーザー/ネオングローをアクセントに使い、夜の繁華街らしい演出を再現して。
```
